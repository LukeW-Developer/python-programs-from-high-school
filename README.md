# python-high-school-projects
This repo contains a collection of programs I made during High School.

The programs featured are:
- A program that can calculate mass, energy or specific latent heat. This was a program I developed in relation to Physics (Science).
- A program that can calculate Compound and Simple Interest. This was a program I developed in relation to Mathematics.
- A program that can calculate hexadecimals to decimal numbers. This was a program I developed in relation to Computer Science.
- Another program that can calculate Decimal number to hexadecimal number. This was a program I developed in relation to Computer Science, this is unfinished.

These are programs that were quickly developed outside of working hours.
